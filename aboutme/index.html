# 深圳计算科学研究院

2021年7月-至今 就职于 深圳计算科学研究院 (opens new window). 研究院下共有3个研究室, 我在IOD(Intelligence Of Data)研究室从事大数据图数据挖掘算法的开发工作, 担任研发工程师一职. 主要开发语言为Python和Go.

IOD研究室主要的产品为: 钓鱼城数据分析系统 (opens new window). "钓鱼城系统"是一个基于图数据的大数据规则挖掘系统. 其挖掘算法核心实现的是本院首席科学家: 樊文飞院士 (opens new window)的相关论文(DGFD, GAR等). "钓鱼城系统"现支持Linux系统分布式云计算服务版本和Window系统单机版本, 当Windows单机版出现计算资源不足的问题时支持用户转换到云计算服务进行计算.

# 项目经历

# 2021年08月
目标

挖掘潜在的储蓄优质客户, 帮助银行完成拉储业务.


成果

使用钓鱼城系统, 围绕存款主题, 利用业务流水建立客户之间的关联, 挖掘出了客户流水与储蓄间的多种拓扑规则.

# 2022年12月
目标

预测携号转网用户群.


成果

使用钓鱼城系统, 挖掘出较为复杂的拓扑结构展现客户之间的个人联系对携号转网的影响, 最终规则准确度可达到30%左右.

# 2023年07月
目标

使用钓鱼城系统, 挖掘潜在的数据规则.


成果

准备交付

# 2023年09月
目标

帮助审计部门预警异常的时序数据, 对各个分行进行预警, 辅助审计部门缩小人工排查范围.


成果

根据无监督异常预警场景, 使用孤立森林和深度学习多种方法, 对时序数据进行离群值判断.
